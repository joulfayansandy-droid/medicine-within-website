Navigation Component

The global navigation system for Medicine Within. This component uses Cinzel for the logo, Raleway for primary navigation links (uppercase, tracked), and Cormorant Garamond for dropdown menu items (italic/serif feel).

Final Item Architecture

  • Offerings (Dropdown: Temple Work, Sacred Feminine, Embodied Tantra, Kambô, Retreats)
  • 1:1 Work (Dropdown: Mentorship, Private Ritual, Archetypal Coaching)
  • Events (/events/index.html)
  • Testimonials (/testimonials/index.html)
  • About (/about/index.html)
  • Contact (/contact/index.html)
