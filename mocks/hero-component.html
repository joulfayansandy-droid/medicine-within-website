MW Hero Component

A highly configurable, visual-first hero section component for Medicine Within. Designed to handle images, videos, and various text placements without blocking critical media elements.

Basic Usage

<section class="mw-hero mw-hero--place-bottom-left mw-hero--overlay-bottom">
    <div class="mw-hero__media">
        <img src="path/to/image.jpg" alt="Description">
    </div>
    <div class="mw-hero__overlay"></div>
    <div class="mw-hero__content">
        <span class="mw-hero__label">Label</span>
        <h1 class="mw-hero__title">Heading</h1>
        <p class="mw-hero__subtitle">Subheading text...</p>
        <div class="mw-hero__cta">
            <a href="#" class="btn">Call to Action</a>
        </div>
    </div>
</section>

Configuration Options

Category Class Modifier Effect
Placement mw-hero--place-center Centered text (Default)
mw-hero--place-bottom-left Text in bottom-left corner
mw-hero--place-split Title at top, CTA at bottom (Face-safe)
Overlays mw-hero--overlay-bottom Black fade at bottom
mw-hero--overlay-teal Branded teal tint
Media mw-hero--media-top Aligns image focus to top
